# Электростатика двух проводящих сфер: метод моментов (MoM) + визуализация

Проект решает **задачу электростатики** для двух идеальных проводников (сейчас — **две сферы**) при заданной разности потенциалов, находит дискретное распределение зарядов на поверхности методом моментов (коллокационная схема), вычисляет потенциал и поле в пространстве и визуализирует 2D-сечение **z = 0** в браузере.

> Важно: визуализация — это **плоский срез** 3D-поля. Линии поля на графике — стримлайны поля **(Ex, Ey)** в плоскости z=0.

## 1. Физическая постановка

Есть два идеальных проводящих электрода (сферы). На них задана разность потенциалов:

\[
\varphi_1 = +\frac{\Delta V}{2},\qquad \varphi_2 = -\frac{\Delta V}{2}.
\]

Нужно найти:

- распр еделение поверхностного заряда на проводниках;
- потенциал \(\varphi(\mathbf r)\) и поле \(\mathbf E(\mathbf r)\) вне проводников;
- полный заряд каждого электрода и ёмкость системы.

Предположения модели:

- среда — вакуум/воздух с \(\varepsilon = \varepsilon_0\);
- электроды — идеальные проводники (эквипотенциальны);
- статический режим (электростатика).

---

## 2. Основные уравнения электростатики

### 2.1. Потенциал точечного заряда

Потенциал в точке \(\mathbf r\) от точечного заряда \(q\), расположенного в \(\mathbf r_0\):

\[
\varphi(\mathbf r) = \frac{1}{4\pi\varepsilon_0}\,\frac{q}{\lVert\mathbf r - \mathbf r_0\rVert}.
\]

Обозначим

\[
K = \frac{1}{4\pi\varepsilon_0}.
\]

В коде:

- `EPS0 = 8.8541878128e-12`
- `K_COULOMB = 1/(4*pi*EPS0)`

### 2.2. Суперпозиция

Для набора зарядов \(\{q_j\}\) в точках \(\{\mathbf r_j\}\):

\[
\varphi(\mathbf r) = K\sum_{j=1}^{N}\frac{q_j}{\lVert\mathbf r - \mathbf r_j\rVert}.
\]

### 2.3. Электрическое поле

\[
\mathbf E(\mathbf r) = -\nabla\varphi(\mathbf r).
\]

Для точечных зарядов поле (векторная форма):

\[
\mathbf E(\mathbf r) = K\sum_{j=1}^{N} q_j\,\frac{\mathbf r - \mathbf r_j}{\lVert\mathbf r - \mathbf r_j\rVert^3}.
\]

## 3. Дискретизация и метод моментов (MoM)

### 3.1. Идея

Идеальный проводник в электростатике должен быть эквипотенциальным: потенциал на его поверхности равен константе \(\varphi = V_k\).

Непрерывно это можно записать как интеграл по поверхности с неизвестной поверхностной плотностью заряда \(\sigma(\mathbf r')\):

\[
\varphi(\mathbf r) = K\int_{S}\frac{\sigma(\mathbf r')}{\lVert\mathbf r - \mathbf r'\rVert}\,dS',\qquad \mathbf r\in S.
\]

Мы заменяем непрерывное распределение заряда дискретным.

### 3.2. Дискретизация поверхности

Каждый электрод разбивается на \(N\) панелей. Для сфер в `main.py` используется равномерное по площади приближение точками алгоритма **Fibonacci sphere**:

- функция `fibonacci_sphere(n, radius, center)` генерирует \(n\) точек на сфере;
- функция `sphere_panels()` задаёт каждой панели одинаковую площадь

\[
S_i \approx \frac{4\pi R^2}{N}.
\]

### 3.3. Коллокационная схема

Каждую панель \(j\) мы аппроксимируем *точечным зарядом* \(q_j\) в её центре \(\mathbf r_j\).

Требование эквипотенциальности накладываем в точках коллокации \(\mathbf r_i\) (в этой реализации это те же центры панелей).

Тогда для каждой коллокационной точки получаем:

\[
\sum_{j=1}^{N_{tot}} K\,\frac{q_j}{\lVert\mathbf r_i - \mathbf r_j\rVert} = V_i.
\]

Это СЛАУ:

\[
A\,\mathbf q = \mathbf b,
\]

где

\[
A_{ij} = K\,\frac{1}{\lVert\mathbf r_i - \mathbf r_j\rVert},
\qquad
b_i = V_i.
\]

В `main.py`:

- матрица строится в `build_mom_system()`
- решение находится `scipy.linalg.solve(A, b)`

## 4. Регуляризация самовзаимодействия (диагональ матрицы)

Для \(i=j\) выражение \(1/\lVert\mathbf r_i - \mathbf r_j\rVert\) сингулярно, т.к. расстояние ноль.

В MoM обычно используют аппроксимацию **самовлияния панели** как потенциал от равномерно заряженного элемента. В этой упрощенной реализации используется *эффективный радиус панели*:

1) Приблизим панель кругом площади \(S_i\):

\[
S_i = \pi a_i^2 \Rightarrow a_i = \sqrt{\frac{S_i}{\pi}}.
\]

2) В коде дополнительно используется масштаб `self_radius_factor`:

\[
\lVert\mathbf r_i - \mathbf r_i\rVert \approx a_i\cdot \text{self\_radius\_factor}.
\]

То есть диагональ матрицы задается как

\[
A_{ii} \approx K\,\frac{1}{a_i\,\text{self\_radius\_factor}}.
\]

Это **регуляризация**, влияющая на точность выполнения граничного условия (потенциала на поверхности). Чем больше панелей и чем точнее self-term, тем лучше эквипотенциальность.

## 5. Вычисление потенциала и поля в произвольных точках

После нахождения \(\mathbf q\) мы вычисляем потенциал и поле в любой точке \(\mathbf r\), используя формулы из раздела 2.

### 5.1. Потенциал

Для набора точек наблюдения \(\{\mathbf r^{(m)}\}\):

\[
\varphi(\mathbf r^{(m)}) = K\sum_{j=1}^{N_{tot}}\frac{q_j}{\lVert\mathbf r^{(m)} - \mathbf r_j\rVert}.
\]

Реализация: `potential_at_points(points_eval, src_points, q)`.

Малое расстояние отсечено, чтобы избегать деления на ноль:

\[
\lVert\cdot\rVert < 10^{-16} \Rightarrow \frac{1}{\lVert\cdot\rVert} \to 0.
\]

(В коде используется `np.inf` в знаменателе.)

### 5.2. Поле

\[
\mathbf E(\mathbf r^{(m)}) = K\sum_{j=1}^{N_{tot}} q_j\,\frac{\mathbf r^{(m)} - \mathbf r_j}{\lVert\mathbf r^{(m)} - \mathbf r_j\rVert^3}.
\]

Реализация: `compute_field(points_eval, src_points, q)`.

## 6. Заряды электродов и ёмкость

Полный заряд каждого электрода:

\[
Q_1 = \sum_{j\in \text{электрод 1}} q_j,\qquad
Q_2 = \sum_{j\in \text{электрод 2}} q_j.
\]

В изолированной системе ожидается:

\[
Q_1 + Q_2 \approx 0.
\]

Ёмкость двухэлектродной системы определяется как:

\[
C = \frac{|Q|}{|\Delta V|},
\]

где \(Q\) — заряд одного электрода (в коде берется \(|Q_1|\)).

## 7. Архитектура проекта

- `main.py`
  - генерация панелей на сферах;
  - сборка матрицы MoM и решение СЛАУ;
  - расчёт потенциала/поля на сетке в плоскости **z = 0**;
  - FastAPI сервер:
    - `GET /` отдаёт `index.html`
    - `POST /api/simulate` выполняет расчёт и возвращает JSON.

- `index.html`
  - UI для параметров (N, радиусы, расстояние, ΔV, размер сетки);
  - теплокарта потенциала (diverging colormap около 0);
  - силовые линии с правильным направлением (стрелки показывают направление поля от + к −);
  - дополнительно запускаются линии, которые **входят в отрицательный электрод со всех сторон**, включая правую (для наглядности), но стрелки всё равно показывают физическое направление E.

- `verify_physics.py`
  - набор проверок: зарядовый баланс, закон Гаусса, оценка эквипотенциальности.

## 9. Физические проверки и ожидаемое поведение

### 9.1. Закон сохранения заряда

Ожидаем \(Q_1 + Q_2 \approx 0\). В численном решении — близко к нулю с малой погрешностью.

### 9.2. Закон Гаусса

Для замкнутой поверхности \(S\), охватывающей заряд \(Q\):

\[
\oint_S \mathbf E\cdot d\mathbf S = \frac{Q}{\varepsilon_0}.
\]

`verify_physics.py` интегрирует поток через контрольную сферу вокруг каждого электрода.

### 9.3. Эквипотенциальность поверхности

Проверяем, что потенциал на коллокационных точках, лежащих на поверхности каждого электрода, близок к целевому \(\pm\Delta V/2\).

## 10. Ограничения и типичные источники погрешности

1) **Self-term (диагональ MoM)** в этой версии приближённый. Он влияет на точность потенциала на поверхности.
2) Коллокация точечными зарядами — упрощение. При малом числе панелей поле рядом с поверхностью может быть «шумным».
3) Визуализация показывает **срез z=0** 3D-поля. Некоторые 3D-инварианты нельзя проверять 2D-операторами (например, \(\nabla\cdot\mathbf E=0\) требует 3 компонент).
4) Если сферы близко (d ≈ r1+r2), задача становится численно жёстче, растёт обусловленность матрицы.
